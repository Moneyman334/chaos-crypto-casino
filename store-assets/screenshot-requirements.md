# App Store Screenshot Requirements

## Google Play Store

### Required Sizes
- **Phone:** 1080 x 1920 px (minimum 2 screenshots, maximum 8)
- **7-inch Tablet:** 1920 x 1200 px (optional)
- **10-inch Tablet:** 2560 x 1800 px (optional)

### Content Guidelines
1. Show actual app functionality
2. No mockup devices or frames
3. Use real content, not placeholder
4. Highlight key features

### Recommended Screenshots (in order):
1. **Main Dashboard** - Show portfolio overview, multi-chain support
2. **Trading Interface** - Margin trading with leverage controls
3. **NFT Marketplace** - Browse and trade NFTs
4. **Copy Trading** - Trader leaderboard and follow system
5. **Wallet Management** - Multi-chain wallet interface
6. **DeFi Features** - Staking pools and yield farming
7. **Token Creator** - One-click deployment interface
8. **Security Dashboard** - Wallet health and monitoring

## Apple App Store

### Required Sizes (iPhone)
- **6.7" Display (iPhone 15 Pro Max):** 1290 x 2796 px (minimum 3, maximum 10)
- **6.5" Display (iPhone 14 Plus):** 1284 x 2778 px (fallback)
- **5.5" Display (iPhone 8 Plus):** 1242 x 2208 px (legacy support)

### Required Sizes (iPad)
- **12.9" iPad Pro:** 2048 x 2732 px (minimum 3, maximum 10)
- **iPad (6th gen):** 1668 x 2224 px (fallback)

### Content Guidelines
- Show actual UI, not simulated
- No customer testimonials in screenshots
- No pricing or promotional text
- Must be appropriate for age rating (17+)

### Recommended Screenshots (in order):
1. **Hero Screen** - Main dashboard with "Web3 Blockchain Empire" branding
2. **Trading Platform** - Professional margin/futures interface
3. **Copy Trading** - Social trading features
4. **NFT Creation** - Token/NFT deployment tools
5. **Multi-Chain Wallet** - 22 network support visual
6. **DeFi Ecosystem** - Staking, farming, DAO treasury
7. **Security Features** - Fraud detection, wallet monitoring
8. **E-Commerce** - Crypto payment acceptance

## Feature Graphic (Google Play Only)
- **Size:** 1024 x 500 px
- **Format:** PNG or JPG (no transparency)
- **Content:** Eye-catching banner showcasing app brand and key value proposition

## App Preview Video (Apple - Optional but Recommended)
- **Length:** 15-30 seconds
- **Orientation:** Portrait for iPhone, Landscape for iPad
- **Content:** Quick demo of core features
- **Format:** MP4 or MOV

## Icon Requirements (Both Stores)
- **Size:** 1024 x 1024 px
- **Format:** PNG (24-bit, no alpha/transparency)
- **Content:** Already created! Use existing cyberpunk-themed icon

## Tools for Creating Screenshots
1. **Browser DevTools** - Set device to iPhone 15 Pro Max (1290x2796)
2. **Chrome Extension** - "GoFullPage" for long scrolling captures
3. **Design Tool** - Figma/Canva for adding text overlays (optional)
4. **Real Device** - Use iPhone/Android for authentic captures

## Pro Tips
✓ Use consistent branding colors (purple/blue cosmic theme)
✓ Show diverse features across screenshots
✓ Avoid cluttered UI - focus on one feature per image
✓ Test on actual devices before submission
✓ Localize screenshots for target markets (if applicable)
