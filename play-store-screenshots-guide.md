## HOW TO CAPTURE SCREENSHOTS

### Option 1: Android Emulator (Best Quality)
1. Open Android Studio
2. Run your app in Pixel 6 Pro emulator (1080 x 2400)
3. Navigate to each screen
4. Use emulator's camera button to capture
5. Screenshots saved to: `~/Desktop` or `~/Pictures`

### Option 2: Real Device
1. Install app on your Android phone
2. Navigate to each screen
3. Press **Power + Volume Down** to screenshot
4. Transfer files to computer via USB or Google Photos

### Option 3: Replit Webview (Quick)
1. Open your app at your Replit URL
2. Open Chrome DevTools (F12)
3. Click "Toggle Device Toolbar" (phone icon)
4. Set dimensions to 1080 x 1920
5. Navigate to each screen
6. Right-click ‚Üí "Capture screenshot"
